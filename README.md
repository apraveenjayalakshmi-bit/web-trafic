# Website Traffic Analysis

## Overview
This project analyzes website traffic data using Python and Jupyter Notebook. The objective is to understand user activity, identify traffic patterns, and gain insights into visitor behavior through statistical analysis and data visualization.

## Dataset
- Dataset: Website Traffic Analysis Dataset
- Source: Kaggle
- File Format: CSV

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Project Objectives
- Load and explore the website traffic dataset.
- Calculate summary statistics (Mean, Median, Mode).
- Check missing values and data types.
- Analyze website traffic trends.
- Visualize visitor activity using charts.
- Generate actionable business insights.

## Dataset Features
- Event
- Date
- Country
- City
- Artist
- Album
- Track
- ISRC
- Link ID

## Visualizations
- Bar Chart: Top 10 Countries by Website Events
- Line Chart: Website Events Over Time
- Pie Chart: Event Distribution

## Key Findings
- Website traffic is concentrated in a few countries.
- User activity changes over time, indicating peak traffic periods.
- Event distribution highlights the most common user interactions on the website.

## Actionable Insights
1. Focus marketing campaigns on countries with the highest website traffic.
2. Schedule content releases during periods of high visitor activity.
3. Analyze user events to improve website navigation and increase engagement.

## How to Run
1. Clone this repository.
2. Open the notebook in Jupyter Notebook.
3. Install the required libraries:
   ```
   pip install pandas matplotlib
   ```
4. Place the dataset in the project folder.
5. Run all cells in the notebook.
